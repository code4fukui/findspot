# findspot
観光地を探す（京都、odp SPARQLエンドポイント）

## デモ
https://codeforfukui.github.io/findspot/

## 機能
- 現在地周辺の観光地を表示
- 京都市のオープンデータや、opdのSPARQLエンドポイントを活用
- Google Mapsを使用

## 必要環境
- ウェブブラウザ
- Geolocation APIに対応したデバイス（任意）

## 使い方
1. ページを開くと、現在地周辺の観光地が地図上に表示されます。
2. 「最短」をクリックすると、現在地から最も近い観光地にフォーカスされます。
3. 「次へ」「前へ」ボタンで、観光地の一覧を移動できます。

## データ・API
- 京都市オープンデータ
- ODP(オープンデータ流通推進協議会)のSPARQLエンドポイント
