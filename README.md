# findspot

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Explore tourist spots (Kyoto, ODP SPARQL endpoint)

## Demo
https://codeforfukui.github.io/findspot/

## Features
- Searches for tourist spots using SPARQL queries
- Displays the results on a Google Maps interface
- Supports multiple SPARQL endpoints (ODP, Kyoto City)
- Provides a nearest spot finder based on the user's location

## Requirements
- Modern web browser

## Usage
1. Open the [demo page](https://codeforfukui.github.io/findspot/) in a web browser.
2. The map will display tourist spots around the default location (Kyoto).
3. Click the "Nearest" button to find the nearest spot based on your location (requires permission to access your location).
4. Use the "Prev" and "Next" buttons to navigate through the displayed spots.

## Data / API
This project uses the Open Data Portal (ODP) and Kyoto City's SPARQL endpoints to retrieve tourist spot information.
